# bytecrft-month2-project
🌐 ByteCraft IT Solutions – Company Portfolio Website (Month 2)

📌 Project Overview

This project was developed during my Month 2 Internship at ByteCraft IT Solutions as a Web Developer Intern.

The objective was to enhance a company portfolio website by improving UI/UX, responsiveness, and adding interactive features to simulate a real-world business website.

🚀 Live Repository

🔗 https://github.com/sachin-rajule/bytecrft-month2-project

✨ Features

 Modern and responsive UI design
 Fully designed Home page with hero section
 Services section with structured layout
 Portfolio section showcasing projects
 Contact form with JavaScript validation
 Mobile-friendly and responsive design
 Clean navigation bar and smooth layout
 
🛠️ Tech Stack

Frontend: HTML5, CSS3, JavaScript

Styling: CSS Flexbox & Grid

Validation: JavaScript
